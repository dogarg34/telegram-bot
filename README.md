# OTP Bot Telegram

A Telegram bot that monitors APIs and forwards OTP messages to Telegram groups.

## Features
- Auto-detect OTP from messages
- Country detection with flags
- Phone number masking
- Duplicate message prevention

## Deployment
1. Add BOT_TOKEN in bot.py
2. Add GROUP_IDS
3. Deploy on Heroku/Railway

## Environment Variables
None required (tokens hardcoded in bot.py)
